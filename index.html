Summary: Latitude vs. X

The purpose of this project was to analyse weather trends based on a location's distance from the equator. I used the OpenWeatherMap API to assemble a weather dataset for over 500 cities across the globe.

Using pandas and matplotlib Python libraries, I plotted four weather variables against latitude: cloudiness, humidity, maximum temperature, and wind speed. This site provides the raw dataset, the plots created, and explanations of trends and correlations observed.
